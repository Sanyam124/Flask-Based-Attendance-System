System Logic & Data Integrity

Logical Hierarchy

  • Classes: The master container for students.
  • Subjects: Academic units taught within classes.
  • Teachers: Assigned to a Class + Subject pairing.
  • Attendance: Tied to the specific Class/Subject/Session session.

Deactivation vs. Deletion

  • Deactivate: Keeps all attendance history. User/Class is hidden but data is safe. (Recommended for past sessions)
  • Delete: Permanent removal. Only possible if no attendance history exists. (Use only for mistakes)
